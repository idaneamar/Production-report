העלאת מחשבון האריזה לאינטרנט (GitHub) — קישור אחד לכל המחשבים
================================================================

המטרה: כתובת אינטרנט אחת שכל המחשבים פותחים. כשמעדכנים את הקובץ פעם אחת —
כולם מקבלים את העדכון אוטומטית ברענון. בלי להעתיק קבצים לכל מחשב.

מה יש בתיקייה הזו ("אתר - להעלאה לגיט"):
- index.html      ← המחשבון (זה אותו קובץ, רק בשם index כדי שהכתובת תהיה נקייה)
- settings.json   ← ההגדרות המרכזיות (לקוחות + מוצרים)


שלב 1 — פתיחת חשבון GitHub (פעם אחת, חינם)
-------------------------------------------
1. היכנס ל-  https://github.com/signup
2. הזן אימייל, סיסמה ושם משתמש (שם המשתמש יופיע בכתובת — בחר שם קצר באנגלית).
3. אשר את האימייל.


שלב 2 — יצירת מאגר (Repository)
--------------------------------
1. בפינה ימין למעלה לחץ על  +  ובחר  "New repository".
2. Repository name:  trili   (או כל שם באנגלית).
3. בחר  Public.
4. לחץ  "Create repository".


שלב 3 — העלאת הקבצים
---------------------
1. בעמוד המאגר לחץ  "Add file"  ←  "Upload files".
2. גרור לתוך החלון את שני הקבצים:  index.html  ו-  settings.json
3. למטה לחץ  "Commit changes".


שלב 4 — הפעלת האתר (GitHub Pages)
----------------------------------
1. במאגר לחץ על  "Settings"  (למעלה).
2. בתפריט הצד לחץ  "Pages".
3. תחת  "Build and deployment" → Source:  "Deploy from a branch".
4. Branch: בחר  "main"  ותיקייה  "/ (root)"  ולחץ  "Save".
5. המתן כדקה ורענן. למעלה תופיע הכתובת, למשל:
      https://השם-שלך.github.io/trili/
6. זו הכתובת לשיתוף! שלח אותה בוואטסאפ/מייל לכל מי שצריך.
   כל אחד פותח אותה בדפדפן — בלי התקנה.


איך מעדכנים אחר כך (לכולם בבת אחת)
-----------------------------------
שינוי במחשבון (קוד): מחליפים את index.html במאגר —
   "Add file" → "Upload files" → גוררים index.html חדש → "Commit changes".

שינוי בהגדרות (לקוחות/מוצרים) — שתי דרכים:
  א. במחשבון: לשונית "הגדרות" → עורכים → "ייצוא הגדרות" → נשמר settings.json →
     מעלים אותו למאגר במקום הקיים (Upload files → Commit). כל המחשבים יתעדכנו.
  ב. ישירות ב-GitHub: נכנסים לקובץ settings.json במאגר → לוחצים על העיפרון (Edit) →
     משנים → "Commit changes".

אחרי כל עדכון — כל אחד פשוט מרענן את העמוד (F5) ומקבל את הגרסה החדשה.
כשהמחשבון פתוח מהכתובת ולא כקובץ, יופיע למעלה התג "⬤ הגדרות מרכזיות".


הערה: אם תעדיף, אפשר גם דרך נטליפיי (netlify.com/drop) — גוררים את התיקייה ומקבלים
כתובת מיד בלי חשבון, אבל לעדכון צריך לגרור שוב. GitHub עדיף לניהול שוטף.
